# 🚀 Government Yojana Management System

A full-stack web application designed to provide structured and easy access to government schemes. The platform helps users explore, filter, and understand schemes efficiently based on categories and requirements.

---

## 📖 Description
This project solves the problem of scattered and unorganized information about government schemes by providing a centralized and user-friendly platform. It enables users to browse schemes, view details, and filter based on categories.

---

## 🚀 Features
- Browse and explore various government schemes  
- Category-based filtering for easy navigation  
- Dynamic content rendering using PHP backend  
- Efficient data handling using MySQL database  
- Clean and user-friendly interface  

---

## 🛠 Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL
